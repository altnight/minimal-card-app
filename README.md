# minimal-card-app

## Feature

- Board, List, Card, Comment, Checklist
- Drag and drop
- Markdown
- Editable content

### Not Feature

- Multi User
- Calendar
- Attach Files, Label, Move, Copy, Archive, and ...
- Storage

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# dummy data
npm run gendata
```
